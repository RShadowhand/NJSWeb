# NJSWeb
Async web server written in Node.JS.
Examples are in "www" folder.

# Features:
* Async file serving
* Using .njs files for dynamic pages
* Templates!
* GET/POST Support

# Requirements:
None

# How to run
1) Install Node.JS

2) Clone the repo.

3-a) Put files in www/

3-b) Change port (Optional)

4) > node webserver.js

5) Navigate to http://localhost:port/

6) Done!
