# NJSWeb
Async web server written in Node.JS.
Examples are in "www" folder.

# Features:
* Async file serving
* Using .njs files for dynamic pages
* Templates!
* GET/POST Support

# Requirements:
None

# How to run
1) Install Node.JS

2) > node webserver.js

3A) Put files in www/

3B) Change port (Optional)

4) Navigate to http://localhost:port/

5) Done!
